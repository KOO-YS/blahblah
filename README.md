# BlahBlah

```
온라인 영어 회화 교육 웹사이트

시간과 장소에 구애받지 않는 편리한 학습 방식
실시간으로 진행되는 1:1 맞춤형 수업을 진행하여 영어 스피킹 능력을 향상
```
